# Developer-Salary-MLP-RNN-Comparison
Implementation and comparison of MLP, RNN, and Regression models for salary prediction using Stack Overflow Survey data (2021-2024) . This project evaluates deep learning architectures against standard regression.


Salary Prediction: MLP vs. RNN
This project explores salary prediction using advanced neural network architectures.

Key Features:

Models Trained: Multilayer Perceptron (MLP) and Recurrent Neural Networks (RNN).

Comparative Analysis: Performance comparison between deep learning models and traditional regression.

Dataset: Stack Overflow Annual Developer Surveys from 2021 to 2024 .
